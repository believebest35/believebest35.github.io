Leetcode 60 第k个排列

leetcode 60

题目描述

给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

  1. “123”
  2. “132”
  3. “213”
  4. “231”
  5. “312”
  6. “321”

给定 n 和 k，返回第 k 个排列。

说明：

  • 给定 n 的范围是 [1, 9]。
  • 给定 k 的范围是[1, n!]。

示例

输入: n = 3, k = 3
输出: “213”

输入: n = 4, k = 9
输出: “2314”

解法

依次固定每一位的取值即可,注意是从0开始计数的因此k需要减1，当k到0的时候就可以直接将剩下的数字直接接在最后。
