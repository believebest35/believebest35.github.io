参照廖雪峰的Java教程进行学习。

JDK和JRE

  • JDK：Java Development Kit

  • JRE：Java Runtime Environment

JAVA_HOME的bin目录

  • java：这个可执行程序其实就是JVM，运行Java程序，就是启动JVM，然后让JVM执行指定的编译后的代码；
  • javac：这是Java的编译器，它用于把Java源码文件（以.java后缀结尾）编译为Java字节码文件（以.class后缀结尾）；
  • jar：用于把一组.class文件打包成一个.jar文件，便于发布；
  • javadoc：用于从Java源码中自动提取注释并生成文档；
  • jdb：Java调试器，用于开发阶段的运行调试。

第一个Java程序

  • 定义了一个公开类Hello（为了规范，类名首字母一般大写）。

  • 程序从名为main()的公开静态方法开始进入（一般成员函数名称首字母小写）。

  • System.out.println()向终端进行打印。

一些Java程序基础

以第一个Java程序为例，定义了名为Hello的公开类，定义了名为main的公开静态方法，即程序入口。

注释

Java的三种注释

其中最后的一种特殊的多行注释格式一般写在类和方法的定义处，用于自动创建文档。

Mac下eclipse按⌘+⇧+F（Windows下按Ctrl+Shift+F）可以格式化代码。

变量

基本类型

  • 整数类型：byte，short，int，long
  • 浮点数类型：float，double
  • 字符类型：char
  • 布尔类型：boolean

大小

整型，与C++不同，Java只有带符号的整型数，最高位为为符号位（0正1负），表示范围：

  • byte：-128 ~ 127 （1字节）
  • short: -32768 ~ 32767 （2字节）
  • int: -2147483648 ~ 2147483647 （4字节）
  • long: -9223372036854775808 ~ 9223372036854775807 （8字节）

浮点型

  • float：4字节，结尾加f进行标志
  • double：8字节

字符型

使用单引号

boolean型

只表示true或者false。JVM一般将其用4个字节进行存储。（同C++，和int互相转）。

final和var

使用final可以定义常量类型，不可以被修改（类似C++ const），使用var可进行自动类型推断（类似C++ auto）。

运算

  • + - * / % ++ -- += -= /= *=
  • > < == >= <= != && || !
  • >> << >>> 最后一个为无符号右移，最高位补0
  • & | ! ^
  • =
  • b ? x : y

类型转换

自动的类型转换都是从小范围到大范围，强转类似C++。

String

数组

与C++数组类型不同，Java的数组类型需要动态声明大小，为引用类型。

采用类型[] 数组名的形式进行声明。
